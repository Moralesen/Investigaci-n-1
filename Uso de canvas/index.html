¿Qué es Canvas?


Es un elemento incluido en html 5, como una especie de lienzo que nos permitira dibujar graficos y realizar animaciones mediante el uso de JavaScript.


El elemento Canvas lo podemos incluir dentro de nuestra página, mediante la etiqueta


Atributos principales de la etiqueta Canvas:


  • Width: Indica el ancho del Canvas
  • Height: Indica la altura
  • Id: Identificador del canvas

El tamaño por defecto del canvas, si no se le proporciona un tamaño específico, es de 300px de ancho por 150px de alto.


Cuando se crea un elemento canvas, por defect este no tendrá contenido ni borde, ya que el contenido se le especificará con código Javascript y para el borde podemos utilizar estilos CSS.
